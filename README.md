# Image_Filter

객체지향 프로그래밍 수업: 팀프로젝트

## 실행 방법

```
$ make
```

```
$ ./IMAGE_FILTER
```

## 프로그램 기능

- 사진에 필터 적용
- 커스텀 필터 제작 가능
- 사진 촬용 기능
- 필터 적용한 사진 저장
